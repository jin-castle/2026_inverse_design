### 2. For each fixable example, create a fixed script
Write the fixed script to a .py file (UTF-8), then docker cp to container.
Every fixed script MUST include:

`python
# -*- coding: utf-8 -*-
import matplotlib
matplotlib.use('Agg')  # Must be before pyplot
import matplotlib.pyplot as plt
import sys, os

# Fix __file__ issue
if not hasattr(sys.modules['__main__'], '__file__'):
    sys.modules['__main__'].__file__ = '/tmp/fixed_{id}.py'

# Hook plt.show() to save instead
_fig_count = [0]
_orig_show = plt.show
def _save_show(*args, **kwargs):
    fname = f'/tmp/kb_results/fixed_{id}_{_fig_count[0]:02d}.png'
    plt.savefig(fname, dpi=80, bbox_inches='tight')
    _fig_count[0] += 1
plt.show = _save_show

# [corrected simulation code here]
`
